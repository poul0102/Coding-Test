## 배열 순회 풀이
```
import java.util.*;

class Solution {
    public int[] solution(int[] progresses, int[] speeds) {
		List<Integer> list = new ArrayList<>(); // 배포 개수 저장 리스트

		int len = progresses.length;
		int[] complete = new int[len];  // 각 기능 완료까지 소요되는 일수

		// 각 기능 완료 일수 계산
		for (int i = 0; i < len; i++) {
			int com = (100 - progresses[i] + speeds[i] - 1) / speeds[i];
			complete[i] = com;
		}

		int cur = complete[0]; // 현재 배포 기준이 되는 작업 완료일
		int count = 1; // 현재 배포에 포함되는 기능 수

		for (int i = 1; i < len; i++) {
			// 현재 기능이 기준 기능보다 빨리 끝나거나 같으면 같이 배포
			if (complete[i] <= cur) {
				count++;
			} else {
				// 지금까지의 기능 수 저장 및 새로운 배포 시작
				list.add(count);
				cur = complete[i];
				count = 1;
			}
		}

		// 마지막 배포 개수 저장
		list.add(count);

		int[] answer = new int[list.size()];

		for (int i = 0; i < list.size(); i++) {
			answer[i] = list.get(i);
		}

		return answer;
	}
}
```


## 큐(Queue)를 활용한 풀이
```
import java.util.*;

class Solution {
    public int[] solution(int[] progresses, int[] speeds) {
        List<Integer> list = new ArrayList<>(); // 배포 개수 저장 리스트
        Queue<Integer> q = new LinkedList<>();  // 각 기능 완료까지 소요되는 일수
        
        // 각 기능 완료 일수 계산
        for (int i = 0; i < progresses.length; i++) {
            int complete = (100 - progresses[i] + speeds[i] - 1) / speeds[i];
            q.offer(complete);
        }

        while(!q.isEmpty()) {
            int cur = q.poll(); // 현재 배포 기준이 되는 작업 완료일
            int count = 1;  // 현재 배포에 포함되는 기능 수
            
            // 현재 기능이 기준 기능보다 빨리 끝나거나 같으면 같이 배포
            while(!q.isEmpty() && q.peek() <= cur) {
                q.poll();
                count++;
            }
            
            // 배포 개수 저장
            list.add(count);
        }
        
        int[] answer = new int[list.size()];
        
        for (int i = 0; i < list.size(); i++) {
            answer[i] = list.get(i);
        }
        
        return answer;
    }
}
```
